# mountain-car

Hand gesture control of MountainCar-v0 from Open-AI (https://gym.openai.com/envs/MountainCar-v0/).

**Requirements:**
1. Suggested to run from Ubuntu 18.04.
2. Python3.6
3. torch==1.5.1

To run an experiment run:

```
python3.6 MCar_test.py
```
